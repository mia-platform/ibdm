# Installations

`ibdm` can be installed in different ways, you can choose the one that better fits your needs and
the operating system you are using:

- [Go](#go)

## Go

If you have [Golang] installed with a version >= 1.13 in your system and you have the `$GOPATH`env
set, you can install `ibdm` like this:

```sh
go install github.com/mia-platform/ibdm@main
```

Or like this if the `install` command is not available

```sh
go get -u github.com/mia-platform/ibdm@main
```
